# Architecture

## Purpose

存放技术规格文档，记录正式设计决策。

## Classification Principle

每份规格以 `YYYY-MM-DD-{描述}.md` 命名，按时间线浏览，不按主题分子目录。

## Direct Children

此目录目前无子目录。

| Child | Definition | Boundary | Excludes |
|---|---|---|---|

## Progressive Disclosure

文件名即摘要。按日期排序即可了解演进历史。

## Change Rules

不为单一主题建子目录。规格数量超过 20 份后再考虑按阶段分组。
