# Architecture

## Purpose

This directory stores raw source material and source tracking data.

## Classification Principle

Raw material is separated from generated wiki knowledge so provenance remains inspectable and generated claims can be traced back to evidence.

## Direct Children

This directory currently has no child directories.

| Child | Definition | Boundary | Excludes |
|---|---|---|---|

## Progressive Disclosure

The top level of `raw/` may hold a small number of source files directly. When sources grow, introduce child directories by source type, domain, project, or ingestion pipeline, and define those boundaries here before moving files.

## Change Rules

Add a child directory only when multiple sources share a durable classification that improves retrieval or provenance tracking.

Do not classify raw sources by generated wiki categories unless that same boundary is meaningful for source management.
