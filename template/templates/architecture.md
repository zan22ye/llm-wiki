# Architecture

## Purpose

This directory stores reusable markdown templates for creating and maintaining LLM Wiki files.

## Classification Principle

Templates are grouped together because they define file contracts rather than domain knowledge or source material.

## Direct Children

This directory currently has no child directories.

| Child | Definition | Boundary | Excludes |
|---|---|---|---|

## Progressive Disclosure

The top level lists core v1 templates. Future child directories may group templates by workflow only when the number of templates grows enough to require subdivision.

## Change Rules

Add new templates when a repeated file type needs a stable contract.

Do not add example content here unless it is part of a reusable template.
